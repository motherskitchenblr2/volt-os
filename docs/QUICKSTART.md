# Quickstart — VOLT OS

Get up and running in 5 minutes.

## Prerequisites

- Node.js 20+
- pnpm 8+

## Install

```bash
git clone https://github.com/New-Code-Garage/VOLT-OS_v1.git
cd VOLT-OS_v1
pnpm install
```

## Run Demo

```bash
pnpm demo
```

## Run Tests

```bash
pnpm test
```

## Start Development

```bash
# Start backend
cd packages/sdk
pnpm dev

# Start Mission Control
cd packages/mission-control
pnpm dev
```
