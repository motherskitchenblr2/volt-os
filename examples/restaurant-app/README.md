# Restaurant App — Vertical Slice Example

Complete example: generate a restaurant management application.

## What This Does

1. Creates a project
2. Research Agent generates requirements
3. Architect Agent creates system design
4. Frontend Agent generates Next.js code
5. QA Agent validates the build
6. Artifacts stored in Memory
7. Download generated project

## Run

```bash
cd examples/restaurant-app
npm install
npm start
```

## Output

- `output/requirements.md`
- `output/architecture.md`
- `output/adr-001.md`
- `output/code/` (Next.js application)
